# gengen #
A browser-based random text snippet generator generator.

## dev dependencies & testing ##
- `haml` with `redcarpet`
- `sass`
- `curl` (to retrieve runtime & testing deps)

run `make test` before viewing `test/test.html`.

## other dependencies ##
- `python2` (for the "server" `make` task)
